# LMDAM

El siguiente repositorio contiene trabajos de Lenguaje de Marcas (LM), una asignatura impartida en 1° de DAM, un FP, o estudios vocacionales.

Además hay una web en la que navegar por los distintos projectos. [Enlace](https://masakk1.github.io/LMDAM/).

## Tecnologías empleadas

La web utiliza [Simple.css](https://simplecss.org/) y HTML tradicional.

La web está hosteada con **Github Pages**.

Cada projecto hace uso de sus propias herramientas.

## Cómo entregar ejercicios

> :clipboard:
> Cada ejercicio nuevo va en una branch distinta

En cada entrega:
1. Crear una PR 
2. Enviar con la siguiente plantilla
3. Esperar hasta una repuesta/review
4. Si no hay respuesta/review, mezclar después de 2 semanas la PR igualmente

Plantilla de entrega
```md
### Entrega de Bootstrap - Práctica3 UT 3 / CSS
- [Repositorio](https://github.com/masakk1/LMDAM)
- **PR**: ____ENLACE A LA PR____
- Está en la rama `RAMA`.
- Por favor, hacer **comentarios en la Pull Request.**
- **Enlace alternativo**: ____ENLACE AL CÓDIGO____
```
